# MxL-GEN

MxL-GEN is a Python framework for **generative engineering workflows** combining:

- Ground truth simulation
- Surrogate machine-learning models
- Black-box optimisation
- Data analysis and visualisation

The goal is to efficiently explore design spaces where direct simulation is expensive by
learning surrogate models and iteratively improving candidate designs.

This documentation is written as a **wiki**:
- Conceptual explanations
- Practical usage guides
- Automatically generated API documentation from source code

---

## What problems does MxL-GEN solve?

- Expensive simulation-based optimisation
- High-dimensional parameter spaces
- Coupling ML surrogates to physics-based models
- Reproducible optimisation experiments
